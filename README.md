# Data Glow

Data Glow é um projeto de portfólio que une beleza, dados e tendências de mercado.

A proposta é analisar produtos de cosméticos, skincare e k-beauty para gerar insights sobre:
- avaliações de consumidores
- ingredientes
- preços
- marcas
- tendências do mercado de beleza

## Objetivo inicial
Investigar quais fatores podem estar relacionados a melhores avaliações de produtos de beleza com base em dados públicos.

## Próximos passos
- selecionar dataset
- limpar e preparar os dados
- realizar análise exploratória
- gerar insights visuais
- publicar resultados
